# Лабораторна робота №1: Семантична мережа та аналіз тексту

Даний проєкт присвячений вивченню методів представлення знань та лінгвістичного аналізу в інтелектуальних системах підтримки прийняття рішень (ІСППР).

## 🚀 Функціонал
Проєкт реалізований у вигляді єдиної HTML-сторінки, що містить:
1.  **Інтерактивну семантичну мережу** — побудовану за допомогою `Google Chart Tools API (Word Tree)`, яка відображає ієрархію технологій веб-розробки.
2.  **Модуль семантичного аналізу** — скрипт на JavaScript, що виконує точний пошук і підрахунок ключових слів «C#» та «Samsung» у заданому тексті з урахуванням контексту.

## 🛠 Технології
* **HTML5 / CSS3** — структура та стилізація звіту.
* **JavaScript (ES6)** — логіка аналізу тексту та робота з регулярними виразами.
* **Google Charts API** — візуалізація даних.

## 📈 Як працює аналіз
Для коректного пошуку мови програмування **C#** використано регулярний вираз із механізмом *lookahead*, що дозволяє точно ідентифікувати символ `#` та уникати помилок при підрахунку. Аналіз проводиться окремо для кожного терміна натисканням відповідних кнопок.
